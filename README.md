# Corpus

A free data science learning platform — clean, fast, and built for depth. Covers Python, Machine Learning, Deep Learning, NLP, Statistics, Visualization, and SQL.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- MDX (Contentlayer)
- Framer Motion
- Deployed on Vercel

## Subjects
- Python
- Machine Learning
- Deep Learning
- NLP
- Statistics
- Visualization
- SQL

## Getting Started

Clone the repo and run locally:

    git clone https://github.com/HoodaVishal/Corpus.git
    cd Corpus
    npm install
    npm run dev


## Project Structure

    src/app/          → All pages and routes
    src/components/   → Reusable UI components
    src/content/      → MDX lesson files
    src/lib/          → Utility functions
    src/styles/       → Global styles and tokens
    src/types/        → TypeScript interfaces
